## Standardized Presentation / Report Requirements (English Version)

### 1. Introduction

* Introduce the research paper or topic.
* Present the machine learning problem clearly:

  * What are the input features?
  * What is the output/target variable?
* Explain the importance and practical applications of the problem.
* Describe the main challenges of the problem.

---

### 2. Related Work and Existing Approaches

* Review previous studies or existing solutions related to the dataset or problem.
* Discuss what has already been done by other researchers.
* Compare different approaches used in prior works.

---

### 3. Dataset Description and Exploratory Data Analysis

* Introduce the dataset:

  * Source of the dataset
  * Number of samples and features
  * Feature types (numerical, categorical, missing values, etc.)
* Perform descriptive statistical analysis:

  * Mean, median, quartiles, variance, standard deviation
  * Distribution analysis
  * Correlation analysis
  * Statistical tests (if applicable)
* Explain the data preprocessing techniques used for training the following models:

  * Decision Tree
  * Naive Bayes
  * Logistic Regression
  * K-Nearest Neighbors (KNN)
  * Random Forest
  * XGBoost

Possible preprocessing techniques may include:

* Handling missing values
* Encoding categorical variables
* Feature scaling / normalization
* Outlier detection and treatment
* Feature selection
* Data balancing techniques
* Train-validation-test splitting

---

### 4. Methodology

* Present the proposed experimental methodology.
* Explain the machine learning algorithms used:

  * Decision Tree
  * Naive Bayes
  * Logistic Regression
  * KNN
  * Random Forest
  * XGBoost
* Describe the training workflow and evaluation process.
* Explain why each method was selected.

---

### 5. Experimental Setup

* Present the hyperparameter configurations for each model.
* Describe the validation strategy:

  * Train/test split
  * Cross-validation method
* Specify the hardware and software environment used for training:

  * CPU/GPU
  * RAM
  * Libraries/frameworks

---

### 6. Experimental Results

* Present the evaluation metrics:

  * Accuracy
  * Precision
  * Recall
  * F1-score
  * ROC-AUC
  * Other relevant metrics
* Compare the performance of different models.
* Use tables, charts, or confusion matrices where appropriate.

---

### 7. Discussion and Analysis

* Analyze and interpret the experimental results.
* Explain why certain models performed better or worse.
* Discuss the impact of preprocessing techniques on model performance.
* Identify strengths, limitations, and possible improvements.
* Highlight the key findings and insights obtained from the experiments.

---

### 8. Conclusion

* Summarize the overall study and findings.
* State the most effective preprocessing methods and models.
* Mention future work or possible extensions.

---

## Presentation Constraints

* Maximum: **25 slides**
* The evaluation focuses on:

  * Understanding of the methods
  * Quality of result discussion and interpretation
  * Ability to draw meaningful conclusions from the experiments
